# Master-Prompt für KI

Kopiere den folgenden Block in deine KI (z.B. Claude) und beschreibe danach, was
du bauen willst. Der Prompt erklärt der KI das Protokoll und die Konventionen
dieses Projekts.

## Der Prompt

````text
Du hilfst mir, ein interaktives Touch-Projekt mit einem ESP32 zu bauen.

# Hardware & Firmware
Ein ESP32 liest kapazitive Touch-Pins über die Library "Touch" aus. Pro Pin gibt
es einen Eintrag in einer Config-Tabelle:

    // { Pin, Event-Name, Taste, Threshold }
    TouchConfig config[] = {
      {1, "button_a", 'a', 3000},
      {2, "button_b", 'b', 3000},
    };

Es gibt zwei Modi:
- MODE_KEYBOARD (nur ESP32-S2/S3): der ESP wird zur USB-Tastatur und "tippt" die
  in der Config angegebene Taste, solange der Pin berührt wird.
- MODE_SERIAL (jeder ESP32): der ESP sendet pro Pin und Loop eine JSON-Zeile über
  Serial mit 115200 Baud:

    {"event":"button_a","pin":1,"touched":true,"raw":1234}

  Felder: event (Name aus der Config), pin (GPIO), touched (true/false ob berührt),
  raw (roher Touch-Messwert).

# Was ich von dir brauche
[HIER beschreiben: z.B. "ein Web-Spiel, das auf button_a springt" oder
 "eine Config-Tabelle für 4 Pads als Klavier"]

# Regeln
- Für Web-Frontends: nutze die Web Serial API. Verbinde mit navigator.serial,
  baudRate 115200, lies zeilenweise und JSON.parse jede Zeile. Reagiere auf den
  Flankenwechsel von "touched" (false->true = gedrückt, true->false = losgelassen).
- Web Serial läuft nur in Chrome/Edge und nur über HTTPS oder localhost.
- Halte den Code einfach und kommentiert. Keine externen Frameworks, außer ich
  verlange es.
- Wenn du Firmware-Config änderst: gib die komplette Config-Tabelle aus und sag,
  in welchen Modus (MODE_KEYBOARD / MODE_SERIAL) sie gehört.
````

## Tipp

Für Web-Apps, die auf Tastendrücke reagieren, kannst du auch die
[Serial-Bridge](/bridge) dieses Projekts nutzen: sie liest die JSON-Events und
löst Tastendrücke im Browser-Tab aus. Für systemweite Tastendrücke (z.B. in
Spielen oder Präsentationen) flashe stattdessen die USB-Tastatur-Firmware auf
einen ESP32-S3.
